## Troubleshooting

### Can't Find Send Report Button
- Make sure you're logged in as a **Student**
- Check you're on the **Student Dashboard** (not another page)
- Try scrolling down on the dashboard

### Report Not Appearing in Admin Panel
- Verify you're logged in as **Admin**
- Check Reports tab loads (look for student names)
- Try refreshing the page (reports auto-refresh every 5 seconds)
- Check browser console (F12) for error messages

### Status Won't Update
- Ensure internet connection is stable
- Try refreshing the page
- Check if you're still logged in
- Look at browser console for errors

### Getting Error Message
- Read the error carefully - it tells you what's wrong
- Common: "Please fill in all fields" â†’ Complete title and description
- Common: "Title too long" â†’ Keep under 100 characters
- Common: "Description too long" â†’ Keep under 500 characters
